Que What is DNS?
Ans Domain Name Service is a system that translates domain names into IP addresses.

Que How does DNS work?
Ans DNS works by assigning IP addresses to domain names. When a user enters a domain name in their browser, the DNS server looks up the IP address for that domain name and returns it to the browser.

Que What are Protocols?
Ans Protocols are a set of rules that define how data is transmitted over a network. Protocols define the format of data and the way it is transmitted.

Que What is HTTP?
Ans HTTP is a protocol that defines how data is transmitted over the internet. HTTP is used to transfer data between clients and servers.

Que What is HTTPS?
Ans HTTPS is a secure version of HTTP that uses encryption to protect data from being intercepted or modified by third parties.

Que What is a TCP ?
Ans TCP is a protocol that defines how data is transmitted over a network. TCP is used to transfer data between clients and servers.
